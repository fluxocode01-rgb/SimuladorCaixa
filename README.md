# 🏦 Simulador de Conta Bancária em Java

Este é um projeto didático desenvolvido em Java para simular as operações básicas de uma conta bancária via console. O objetivo é aplicar conceitos fundamentais de **Programação Orientada a Objetos (POO)**.

## 🚀 Funcionalidades

* **Depósito:** Adiciona valores ao saldo com validação.
* **Saque:** Retira valores do saldo, impedindo saques maiores que o valor disponível.
* **Consulta de Extrato:** Exibe o nome do titular e o saldo atualizado.
* **Interface Interativa:** Menu dinâmico via terminal utilizando a classe `Scanner`.

## 🛠️ Tecnologias Utilizadas

* **Java JDK 23** (ou superior)
* **Lógica de Programação**
* **Encapsulamento** (Atributos privados e métodos públicos)
